### [iMathAS-Gradebook-Tracker-Enhanced](https://github.com/petervannord/iMathAS-Gradebook-Tracker-Enhanced)
A custom browser-side enhancement for the iMathAS academic platform.
* **Functionality:** Injects custom tracking logic into the native gradebook to provide deadline countdowns and grade projections.

<summary><b>🖱️ Humanized Stealth Auto-Clicker</b></summary>

### [Humanized-Stealth-Auto-Clicker-Mac](https://github.com/petervannord/Humanized-Stealth-Auto-Clicker---Mac)
A low-level macOS utility focused on anti-detection and system-level event simulation.
* **Humanization Engine:** Implements randomized delay intervals and micro-coordinate "jitter" to bypass heuristic pattern detection.
* **System Integration:** Leverages the **Quartz Event Services** and **Core Graphics** frameworks to interact directly with the macOS window server.
* **Technical Challenge:** Successfully mimics organic hardware interrupts, showcasing proficiency in C++ and system-level API interaction.
</details>
